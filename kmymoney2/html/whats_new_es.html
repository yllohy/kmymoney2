Whats new in 0.8

Personal Finance Software

Return to the Welcome page

Hemos trabajado duro para que esta versión sea mejor y má fácil de usar. Hemos agregado una cantidad de nueva funcionalidad a esta versión y además unas cuantas mejoras. Confiamos en que le gustará lo que vea.

Aquí están algunas de las nuevas funciones en esta versión:

  • Inversiones
  • Reportes
  • Múltiples divisas
  • Conversor de Archivos de GnuCash
  • Importar de OFX
  • Web Connect
  • Un look and feel mejorado -incluyendo un nuevo conjunto de íconos.
  • Codificación del Archivo de Datos
  • Calculadora para ingresar valores
  • Cotizaciones Online para Acciones y Divisas
  • Support for VAT
  • Ability to write importer plugins
  • "Anonymous" File Format to assist debugging

Por favor pruebe las nuevas funciones y haganos saber lo que piensa. Esperamos que disfrute de esta nueva versión de KMyMoney. Aunque hemos hecho lo mejor para eliminar errores en esta versión, es posible que no los hayamos eliminado a todos. Por favor haganos saber de cualquier comportamiento anormal en el programa seleccionando "Reportar error..." del menú de Ayuda o enviando un correo a la lista de correo de desarrolladores. kmymoney2-developer@lists.sourceforge.net
