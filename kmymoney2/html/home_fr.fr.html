Bonjour et bienvenue à KMyMoney2 version 0.5.2

L'équipe de développement est fière de présenter la nouvelle version du logiciel de comptabilité personnelle basé sur le moteur MyMoney.

En sus de nombreuses corrections de bogues, cette version contient les nouvelles fonctionnalités majeures suivantes :

  • Import QIF
  • Support des opérations planifiées
N'hésitez pas à tester les nombreuses nouvelles fonctionnalités vous-même.

Note : Même si cette version peut paraître stable dans une certaine mesure, cela reste une version de développement. N'hésitez pas à nous signaler tout comportement anormal du programme en envoyant un courriel à kmymoney2-developer@lists.sourceforge.net, la liste de discussion des développeurs.

Nous recherchons toujours des traducteurs, et des personnes qui pourraient travailler sur le manuel de cette application. Si vous êtes intéressés, vous pouvez nous aider, et si vous avez d'autres talents pour contribuer à cette application n'hésitez pas pas à contacter les développeurs sur la liste de discussion kmymoney2-developer@lists.sourceforge.net.

Attention : Comme cette version de KMyMoney2 écrit uniquement ses données dans son nouveau format, les fichiers écrits avec cette version ne peuvent être lus par les versions précédentes de la branche stable (cad 0.4 et antérieurs). Si vous voulez malgré tout utiliser une ancienne version de KMyMoney2 avec vos fichiers de données, créez une copie de sauvegarde de vos données comptables. Il n'existe malheureusement pas de solution pour revenir en arrière.
