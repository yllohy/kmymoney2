Bonjour et bienvenue dans KMyMoney version 0.6rc1

L'équipe de développement est fière de vous présenter la version candidate de la prochaine mouture de

KMyMoney, le Gestionnaire de Finances Personnelles pour KDE

De nombreuses nouvelles fonctionnalités ont été ajoutées à cette version, dont voici les plus importantes :

  • saisie des opérations par formulaire
  • vue du registre mise à jour
  • Support pour davantage de types de comptes
  • Support pour les échéances
  • Fonctions de recherches améliorées
  • Assistants pour vous aider à créer et rapprocher vos comptes
  • Support des langues français et allemand
Veuillez essayer vous-même les nombreuses fonctions de KMyMoney. Bien que nous ayons fait de notre mieux pour supprimer les erreurs dans cette version, il se peut qu'il en reste. Veuillez nous faire savoir tout fonctionnement anormal du programme en envoyant un e-mail à kmymoney2-developer@lists.sourceforge.net, la liste de discussion des développeurs.

Attention : Comme cette version de KMyMoney écrit seulement les fichiers dans son nouveau format, les fichiers écrit par cette version ne peuvent plus être lus par les anciennes versions (cad 0.4 et précédentes). Si vous désirez malgré tout utiliser ces versions de KMyMoney avec vos fichiers de données, assurez-vous que vous ayez effectué une copie de sauvegarde de vos données financières. Il n'existe malheureusement de possibilité de retourner en arrière.

Nous recherchons aussi des traducteurs et des personnes pour travailler sur le manuel de l'application. Si vous le voulez, vous pouvez nous aider dans ce domaine, ou si vous avez d'autres talents pour contribuer à cette application, contactez-nous sur la liste de discussion des développeurs à l'adresse kmymoney2-developer@lists.sourceforge.net.
