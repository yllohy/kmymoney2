Whats new in 0.8

Personal Finance Software

Return to the Welcome page

We have been working hard to make this release better and easier to use in every way. We have added a lot of new features to this version and have also made quite a few improvements. We are confident you will like what you see.

Here are some of the new features found in this release:

  • Investments
  • Reports
  • Multiple currencies
  • GnuCash File Converter
  • OFX Import
  • Web Connect
  • An improved look and feel -including a new icon set.
  • Encryption of the Data File
  • Calculator Widget when entering values
  • Online Quotes for Stocks & Currencies
  • Support for VAT
  • Ability to write importer plugins
  • "Anonymous" File Format to assist debugging

Please try the new features and let us know what you think. We hope that you enjoy using this new version of KMyMoney. Even though we have done our very best to eliminate any errors in this version, it is still possible that we did not catch them all. Please let us know about any abnormal behaviour in the program by selecting "Report bug..." from the help menu or by sending an e-mail to the developers mailing list. kmymoney2-developer@lists.sourceforge.net
