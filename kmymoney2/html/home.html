Hello and welcome to KMyMoney version 0.6rc1

The development team is proud to present the first candidate for the next stable release of

KMyMoney, the Personal Finance Manager for KDE

Many new features have been added to this version, containing the following highlights:

  • Form based transaction entry
  • Updated ledger view
  • Support for more account types
  • Support for scheduled transactions
  • Enhanced search functionality
  • Wizards which support you in creating and reconciling your accounts
  • Language support for French and German
Please try the many features of KMyMoney yourself. Even though we have done our very best to eliminate any errors in this version, it is still possible that we did not catch them all. Please let us know about any abnormal behaviour of the program by sending e-mails to kmymoney2-developer@lists.sourceforge.net, the developers mailing list.

Warning: Since this version of KMyMoney only writes data files in it's new format, files written with this version cannot be read by old versions (e.g. 0.4 and prior). If you still want to use these versions of KMyMoney with your data files, please make sure you keep a backup-file of your finance data. Unfortunately, there's no way back.

Also, we are still looking for translators and people who could work on the handbook for this application. If you feel, you could help us out on this or have any other talents you want to contribute to this application, please feel free to contact us using the developers mailing list at kmymoney2-developer@lists.sourceforge.net .
