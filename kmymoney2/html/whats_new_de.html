Dies ist ein Entwicklungs-Release und darum kann es vorkommen, dass einige der neueren Funktionen des Programms noch nicht so tun, wie sie sollen. Releases wie dieses werden auch als 'instabil' bezeichnet; es ist also eine gute Idee, häufig Backups der Daten zu machen. Das gilt auch für die Datei, die mit der aktuellen 0.8.x Version erstellt wurde.
Die meisten der Entwickler vertrauen allerdings ihre Daten dieser Version des Programmes schon für längere Zeit an.

v0.9.2 Release notes:

  • Mehrere Verbesserungen und Ergänzungen für HBCI und OFX import in den Bereichen Transaktionszuweisung und Fehlerbehandlung
  • OFX Unterstützung von Money Market Konten
  • Verschlüsselung für mehrere Schlüssel
  • Neue Berichte: Vorschau, Konto-Stammdaten, Darlehen
  • Neue Konten können direkt als Unterkonten angelegt werden
  • Neue Übersetzungen: Schwedisch, Chinesisch, Türkisch, Rumänisch
  • Anfängliche Unterstützung einer CMake basierten Build-Umgebung
  • Möglichkeit zum Erstellen von Eigenkapital-Konten und Ändern von Buchungen in diesen Konten
  • Direkter Zugriff auf Funktionen für geplante Buchungen aus der Finanzübersicht wie z.B. Ändern, Erfassen und Überspringen
  • Anzeige von geplanten Buchungen im Kontenbuch
  • Verbessertes Layout und mehr Informationen in der Finanzübersicht
  • Unterstützung von mehreren Währungen in Investmentberichten
  • Verbesserte Online Banking Unterstützung
  • Verbessertes Layout der Vorausschau-Ansicht
  • Verbesserte Unterstützung von Datenbanken (Testphase)
  • Verbesserte Dokumentation und README Dateien
  • Viel Code aufgeräumt
  • Ergänzte Übersetzungen: Argentinisch, Deutsch, Holländisch, Portugiesich, Brasilianisch, Italienisch, Französisch, Spanisch
  • Viele Korrekturen in den Programmteilen: Kategorien, Suchen, Kontenbuch, Berichte, Bedienung mit der Tastatur, Taschenrechner, Budgets, Import, Geplante Buchungen, Konten und Darlehen

Probieren Sie die neuen Funktionen aus und teilen Sie uns mit, was sie davon halten. Wir hoffen, dass Sie mit der neuen Version von KMyMoney Spass haben werden. Obwohl wir unser Möglichstes getan haben, um Fehler in dieser Version zu vermeiden, kann es trotzdem vorkommen, dass wir nicht alle gefunden haben. Bitte teilen Sie uns mit, wenn sich das Programm einmal nicht so verhält, wie es soll. Sie können dazu die Funktion "Fehler melden..." aus dem Hilfe-Menu verwenden oder eine e-mail and die Entwickler-Mailing-Liste kmymoney2-developer@lists.sourceforge.net schicken.
