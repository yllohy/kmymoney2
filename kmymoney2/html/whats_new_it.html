Novità della versione 0.7

Software per le finanze personali

Ritorna alla pagina di benvenuto

Abbiamo lavorato duramente per rendere questa versione migliore e più facile in ogni senso. Abbiamo aggiunto molte nuove funzioni a questa versione e abbiamo fatto alcuni miglioramenti. Condfidiamo che ciò che vedrete vi piacerà!

Elenchiamo di seguito alcune nuove funzioni di questa versione:

  • Investimenti
  • Report
  • Valute multiple
  • Convertitore GnuCash
  • Importazione OFX
  • Connessione Web
  • Un look migliorato - incluso un nuovo set di icone.
  • Cifratura del file dei dati
  • Calcolatrice integrata per l'inserimento dei valori
  • Quote online per i titoli di mercato
  • Supporto per VAT
  • Possibilità di scrivere plugin di importazione
  • Formato file "anonimo" per assistere il debug

Vi preghiamo di provare le nuove funzioni e farci sapere la vostra opinione. Speriamo che vi divertitate usando questa nuova versione di KMyMoney. Anche se abbiamo fatto del nostro meglio per eliminare qualsiasi errore in questa versione, è ancora possibile che non li abbiamo trovati tutti. Vi preghiamo di farci sapere qualsiasi comportamento anomalo del programma selezionando "Segnala un bug..." dal menu Aiuto o mandando una e-mail alla mailing list degli sviluppatori. kmymoney2-developer@lists.sourceforge.net
