Novidades na versión 0.8

Software de Finanzas Persoais

Voltar á páxina de benvida

Estivo-se traballando moi duro para facer esta versión mellor e mais fácil de usar no dia a dia. Foron engadidas multiples características novas e fixeron-se abundantes melloras. Confiamos en que goste do resultado.

Estas son algunhas das novas características desta versión:

  • Investimentos
  • Informes
  • Multiples divisas
  • Conversor de arquivos GnuCash
  • Importazón OFX
  • Web Connect
  • Aspeito mellorado -incluindo novas iconas.
  • Cifrado dos Arquivos de Dados
  • Calculadora para a introduzón de importes
  • Cotizazóns online de Aczóns e Divisas
  • Recursos para xestionar o IVE
  • Permite escreber plugins de importazón
  • Formato de Arquivo "Anónimo" para facilitar a depurazón de erros

Prove as novas características e faga-nos chegar a sua opinión. Esperamos que desfrute usando esta nova versión de KMyMoney. Apesar de que fixemos o posíbel para eliminar os erros que pudese haber nesta versión, seguramente quedan alguns. Por favor, faga-nos saber calquer comportamento anormal no programa mediante a opzón "Informar de erro..." do menu axuda ou enviando un e-mail á lista de correo dos desenvolventes. kmymoney2-developer@lists.sourceforge.net
